void UDPManager::connectTo ( std::string  address,
int  port 
)

Send a UDP bridge hello packet to the specified address and port, and creates an interface associated to that peer. The interface is stored in the "connectingLinks", and the router is NOT notified immediately.

No assumptions should be made about the state of the link after this call.
